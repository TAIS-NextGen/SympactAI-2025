To generate roadmaps of other persons, just go to:
.\Robee_Cleaned_Pipeline\MilestonesGeneration\Persons

Add the person name in .\Robee_Cleaned_Pipeline\MilestonesGeneration\Persons\names and its related files, and add their LinkedIn profile as a PDF if it exists in \Robee_Cleaned_Pipeline\MilestonesGeneration\Persons\linkedin

Then run the generate_draftroadmap.py file to get the causal_output.json

If you want to get to the interface part, just run:
streamlit run final_interface.py
